Platform Game

Socket.io Canvas API Node.js MongoDB

📖 Gambaran Proyek

Platform game multipemain di mana pemain dapat bertanding secara real-time dalam berbagai mode permainan. Menggunakan WebSockets untuk komunikasi latensi rendah dan HTML5 Canvas untuk rendering 60fps yang mulus. Dilengkapi dengan sistem pencarian lawan, papan peringkat, dan obrolan dalam game.
